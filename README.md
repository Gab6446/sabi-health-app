# Proactive Voice AI App

This is the codebase for the **Proactive Voice AI App** (Sabi Health), an AI-driven health monitoring companion that issues proactive warnings and collects community health data.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

## Cavista Hackathon 2026: AI Use Disclosure

In compliance with **Rule 5: Responsible Use of AI Tools** of the Cavista Hackathon guidelines, the authors of this project disclose the following:

- **Agentic coding assistance:** The development of this repository was assisted by an AI agent under the direct supervision of the authors to implement UI/UX improvements, code refactoring, and logical scaffolding.
- **AI-generated features:** The generation of proactive voice scripts within the application is simulated using prompts designed for Gemini 1.5 Flash to ensure culturally appropriate health advice.
- **No sensitive data exposed:** All data models and records used during this hackathon are mock data or public data; no private or real patient data was submitted to any AI tool.
